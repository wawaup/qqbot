# 群机器人使用说明

## @机器人 查有货商品

| 指令 | 返回内容 |
|---|---|
| `推荐` / `gpt正价` / `正价冲` | 店铺推荐商品 |
| `gpt` | GPT 成品号 |
| `接码` | 接码服务 |
| `claude` | Claude 商品 |
| `gemini` | Gemini 商品 |
| `grok` | Grok 商品 |
| `苹果id` / `邮箱服务` | 苹果ID / 邮箱类 |
| `清单` / `menu` | 所有分类全部有货商品 |

---

## 直接发关键词（不需要@）

| 关键词 | 回复内容 |
|---|---|
| 店铺链接 / 在哪买 | 店铺地址 |

| 哪里查订单 / 订单卡密 | 订单查询 |
| 售后 / 用不了 | 售后联系 |
